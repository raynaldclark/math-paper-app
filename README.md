# 小学数学试卷生成器

一款面向小学三年级学生的数学练习题生成器，支持生成PDF格式的练习试卷。

## 功能特点

- 支持多种题型：竖式计算、混合运算、单位换算、图形计算等
- 题目根据三年级教学大纲设计
- 可自定义每种题型的数量
- 支持生成PDF格式试卷
- 支持分享试卷

## 构建

本项目使用 GitHub Actions 进行自动化构建，每次推送到 main 分支时会自动构建 APK。

### 手动构建

```bash
flutter pub get
flutter build apk --debug
```

## 题型说明

### 竖式计算
- 竖式加减法（最多五位数）
- 竖式乘法（五位数乘一位数）
- 竖式除法（五位数除以一位数）

### 混合运算
- 加减乘除混合运算
- 带括号混合运算
- 求未知数△

### 单位换算
- 人民币换算（元角分）
- 时间换算（小时分钟秒）
- 长度换算（米分米厘米毫米）
- 质量换算（千克克）

### 图形与测量
- 周长和面积计算
- 面积单位换算

### 概念题
- 找规律填数
- 倍的认识
- 小数加减法
- 方向与位置
